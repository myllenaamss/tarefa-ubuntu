 **MEU SITE TERMINAL**  

**pwd**: Mostra o caminho da pasta atual.

**ls**: Lista arquivos e pastas.

**cd**: Entra em pastas.

**mkdir**: Cria pastas.

**touch**: Cria arquivos.

**cp**: Copia arquivos e pastas.

**mv**: Move ou renomeia arquivos ou pastas.

**rm**: Remove arquivos.

**nano/vim**: Editores de textos internos.
